REST Endpoints Expected:
  GET    /markets                    - List markets
  GET    /markets/:id                - Get market details
  POST   /markets                    - Create market
  POST   /orders                     - Place order
  DELETE /orders/:id                 - Cancel order
  GET    /orders                     - List user orders
  GET    /orderbook/:marketId        - Get order book
  GET    /trades                     - List trades
  GET    /markets/:id/trades         - Market trades
  GET    /positions                  - User positions
  GET    /user/profile               - User info
  GET    /user/balance               - USDC balance
  POST   /user/deposit               - Deposit USDC
  POST   /user/withdraw              - Withdraw USDC
  GET    /markets/:id/price-history  - Price history

WebSocket Channels:
  orderbook:marketId - Order book updates
  trades:marketId    - Trade notifications
  markets:updates    - Market status changes

Environment Configuration:
  NEXT_PUBLIC_API_URL                 - API base URL
  NEXT_PUBLIC_WS_URL                  - WebSocket URL
  NEXT_PUBLIC_STELLAR_HORIZON_URL     - Stellar Horizon endpoint
  NEXT_PUBLIC_STELLAR_NETWORK         - Network passphrase
  NEXT_PUBLIC_USDC_ISSUER             - USDC issuer address
