## Source Code Structure: src/

### Core Types
✓ src/types/index.ts              - All TypeScript interfaces and types
  - Order, Trade, Market, Position, UserBalance
  - Settlement, OrderBook, WebSocket message types

### Matching Engine
✓ src/engine/orderbook.ts         - Single-outcome order book with CLOB
  - Price-time priority matching
  - Support for limit, market, IOC, FOK orders
  - Efficient order book snapshots

✓ src/engine/matching.ts          - Multi-market matching engine
  - Order routing and validation
  - Balance management and locking
  - Position tracking

### Database Layer
✓ src/db/database.ts              - SQLite persistence layer
  - 6 tables: markets, orders, trades, positions, balances, settlements
  - CRUD operations for all entities
  - Transaction support

### API Layer
✓ src/api/routes.ts               - Express REST API endpoints (13 endpoints)
  - Market management (create, list, resolve)
  - Order placement and cancellation

  - Admin endpoints

✓ src/api/websocket.ts            - WebSocket real-time server
  - Channel-based subscriptions
  - Order book updates
  - Trade broadcasts
  - Market status updates

### Settlement Pipeline

## Key Features Implemented

1. CLOB Engine
   - Price-time priority matching
   - FIFO within price levels
   - Support for 4 order types
   - O(log n) matching performance

2. Order Types
   - Limit: Placed on book if no match
   - Market: Immediate best-price fill
   - IOC: Immediate-or-cancel
   - FOK: Fill-or-kill
